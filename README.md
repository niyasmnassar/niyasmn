# Niyas M N — Portfolio

Dark luxury React portfolio with GSAP ScrollTrigger parallax animations.

## Quick Start

```bash
npm install
npm start
```

Open http://localhost:3000

## Build for Production

```bash
npm run build
```

## Notes
- CV file is in `public/` and served at root — download button works automatically.
- GSAP 3.12 + ScrollTrigger loaded from CDN (no extra install needed).
